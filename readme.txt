Enable Accelerated Mobile Pages (AMP) for only POSTS on your WordPress site.

== Description ==

This plugin provides Support mobile optimized content for posts that can load instantly everywhere.

With the plugin active, all posts on your site will have dynamically add the support for AMP `/postamp/` to the end your post URLs. For example, if your post URL is `http://example.com/2018/01/26/helloword/`, you can access the AMP version at `http://example.com/2018/01/26/helloword/postamp`, and you can access through `http://example.com/?p=3&postamp=1`



Note : this plugin only creates AMP content but does not automatically display but only availabe on  Google Search. 

Note : Simplified AMP Page which anyone can use / change the code with minimal knowledge

== Installation ==

1. Upload the folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You may need to refresh your permalinks on settings page
